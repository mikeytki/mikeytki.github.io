常用命令

解压
  • 将文件解压到特定目录，可以使用-C选项。例如，将文件解压到/path/to/directory/目录
    • tar -xzvf XXX.tar.gz -C /path/to/directory/
    • unzip XXX.zip # 解压zip文件
移动

  • 复制 目录cl 到 目录home（～）下

    • cp -r cl home （-r选项用于递归复制，复制整个目录树）

  • 将 文件 a.txt 移动到 目录home下

    • mv a.txt home

  • 将 目录cl 移动到 目录home下

    • mv cl home
重命名

  • 将 a.txt 重命名为 b.txt

    • mv a.txt b.txt

  • 将 目录cl 重命名为 目录cl2

    • mv cl cl2
查看文件和目录

  • 列出当前目录下的所有文件和目录

    • ls -l # 以详细格式列出
    • ls -a # 列出所有文件，包括隐藏文件

  • 查看文件内容

    • cat filename # 显示文件内容
    • less filename # 分页查看文件内容
    • head filename # 查看文件的前10行
    • tail filename # 查看文件的后10行
文件和目录操作

  • 创建目录

    • mkdir new_directory # 创建新目录

  • 删除文件和目录

    • rm filename # 删除文件
    • rm -r directory # 递归删除目录及其内容

  • 复制文件和目录

    • cp filename destination # 复制文件
    • cp -r directory destination # 递归复制目录
权限管理

  • 修改文件权限

    • chmod 755 filename # 修改文件权限
    • chmod -R 755 directory # 递归修改目录权限

  • 修改文件所有者

    • chown user:group filename # 修改文件的所有者和组
系统管理

  • 查看当前用户`

    • whoami # 显示当前用户

  • 查看系统信息

    • uname -a # 显示系统信息
    • top # 实时查看系统进程

  • 查看磁盘使用情况

    • df -h # 显示磁盘使用情况
    • du -sh directory # 显示目录的大小

  • 下载文件

    • wget url # 从指定URL下载文件
其他

  • 查找文件

    • find /path -name filename # 在指定路径查找文件
    • locate filename # 快速查找文件（需要更新数据库）

  • 搜索文本

    • grep 'text' filename # 在文件中搜索指定文本

  • 处理文本文件

    • cut -d':' -f1 filename # 按分隔符切割文件并提取特定字段
    • sort filename # 对文件内容进行排序
    • uniq filename # 去除文件中的重复行
    • wc -l filename # 统计文件的行数
进程管理

  • 查看当前进程

    • ps aux # 显示所有进程
    • top # 实时查看系统进程
    • htop # 更友好的进程管理工具（需要安装）

  • 杀死进程

    • kill PID # 根据进程ID杀死进程
    • killall process_name # 根据进程名称杀死所有相关进程
    • kill -9 name
